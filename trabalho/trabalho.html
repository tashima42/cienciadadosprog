Introduo

O conjunto de dados "Sistema E-Saude - Perfil de atendimento Odontolgico nas Unidades Municipais de Sade de Curitiba" rene informaes sobre atendimentos realizados por profissionais de odontologia em unidades de sade. Esses dados mostram o perfil dos usurios atendidos, com informaes demogrficas, socioeconmicas e de sade, alm de detalhes sobre os procedimentos realizados.

df = df.loc[:, ['Data do Atendimento', 'Data de Nascimento', 'Sexo', 'Descrio do Procedimento', 'Descrio do CBO']].copy()

df = df.copy()

# Remove linhas com NaN
df = df.dropna()

# Transformar em objetos datetime
df['Data do Atendimento'] = pd.to_datetime(df['Data do Atendimento'], format='%d/%m/%Y %H:%M:%S')
df['Data de Nascimento'] = pd.to_datetime(df['Data de Nascimento'], format='%d/%m/%Y %H:%M:%S')
df['Idade'] = (df['Data do Atendimento'] - df['Data de Nascimento']).dt.days // 365

# Teste de Wilcoxon (Mann-Whitney U)
idades_f = df[df['Sexo'] == 'F']['Idade']
idades_m = df[df['Sexo'] == 'M']['Idade']
stat, p = mannwhitneyu(idades_f, idades_m, alternative='two-sided')
# Exibir resultado do teste como tabela
resultado_mannwhitney = pd.DataFrame({
  'Estatstica': [stat],
  'p-valor': [p]
})
display(Markdown("**Tabela 6: Teste de Mann-Whitney U (Wilcoxon) para idade por sexo**"))
display(HTML(resultado_mannwhitney.to_html(index=False)))
