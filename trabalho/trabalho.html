Descrio do dataset

O conjunto de dados "Sistema E-Saude - Perfil de atendimento Odontolgico nas Unidades Municipais de Sade de Curitiba" rene informaes sobre atendimentos realizados por profissionais de odontologia em unidades de sade. Esses dados mostram o perfil dos usurios atendidos, com informaes demogrficas, socioeconmicas e de sade, alm de detalhes sobre os procedimentos realizados.

Este contjunto de dados pode ser encontrado no Portal de Dados Abertos da Prefeitura de Curitiba.

  • Data do Atendimento
  • Data de Nascimento

  • Descrio da Unidade

Uma coluna adicional, "Idade", foi criada a partir da diferena entre a data do atendimento e a data de nascimento.

Linhas com valores nulos e com nmero de cmodos considerados invlidos (acima de 20) foram removidas.

In[4]:
df = df.loc[:, ['Data do Atendimento', 'Data de Nascimento', 'Sexo', 'Descrio da Unidade', 'Descrio do Procedimento', 'Descrio do CBO', 'Energia Eltrica', 'Cmodos', 'Municcio', 'Bairro', 'Nacionalidade']].copy()

df = df.copy()

# Remove linhas com NaN
df = df.dropna()
In[5]:
# Transformar em objetos datetime
df['Data do Atendimento'] = pd.to_datetime(df['Data do Atendimento'], format='%d/%m/%Y %H:%M:%S')
df['Data de Nascimento'] = pd.to_datetime(df['Data de Nascimento'], format='%d/%m/%Y %H:%M:%S')
df['Idade'] = (df['Data do Atendimento'] - df['Data de Nascimento']).dt.days // 365
In[6]:
# Transformar em booleanos
df['Energia Eltrica'] = df['Energia Eltrica'].map({'Nao': False, 'Sim': True})
In[7]:
# Transformar em integer
df['Cmodos'] = df['Cmodos'].astype(int)
# Limpeza de casas com mais de 20 comodos
df = df[df['Cmodos'] <= 20]
